# Research current facts

1. Use Context7 for package and API documentation.
2. Prefer official documentation when Context7 is insufficient.
3. Cite the URL and retrieval date for current-state claims.
4. Do not guess versions, schemas, or deprecation status.
5. Ask before installing packages or adding MCP servers.
