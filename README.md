## Claude Code team setup

This repository includes lightweight shared Claude Code instructions, skills,
and MCP configuration for Context7 and Dart.

On Windows:

```powershell
pwsh -NoProfile -File scripts/setup-claude.ps1
```

Then start Claude Code from the repository root, approve the project MCP
servers, and run `/setup-team`. Optional personal integrations remain
approval-gated and are not committed to the repository.
